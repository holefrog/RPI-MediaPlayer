
# Waveshare WM8960 Sound Board

## 0. 参考资料 (Reference)
该板为 Sound Board，**非 Audio HAT**。  
官网链接：[Waveshare WM8960 Audio Board](https://www.waveshare.com/wm8960-audio-board.htm)

---

## 1. 接线 (Connection)
| PIN           | RPi  | 描述 (Description)           |
|---------------|------|----------------------------|
| 5V            | 2/4  | 5V 电源                     |
| GND           | 6/9  | 接地 (Ground)               |
| SDA           | 3    | I2C 数据线 (Data input)     |
| SCL           | 5    | I2C 时钟线 (Clock input)    |
| CLK           | 12   | I2S 位时钟 (Bit clock input)|
| LRCLK (WS)    | 35   | I2S 帧时钟 (Frame clock input)|
| DAC (RXSDA)   | 40   | I2S 数据输出 (Data output)  |
| ADC (TXSDA)   | 38   | I2S 数据输入 (Data input)   |

> 注意：请正确连接电源、地和数据线，否则可能无法正常使用。

---

## 2. 验证 I2C (Verify I2C)

### 安装工具
```bash
sudo apt update
sudo apt install i2c-tools
```

### 列出 I2C 设备
```bash
ls /dev/i2c-*
i2cdetect -y 1
```

示例输出：
